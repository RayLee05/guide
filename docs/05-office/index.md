# Office 办公软件

## 为什么需要学 Office？

大学四年你几乎每天都要用到 Office：
- **Word** — 写论文、写报告、写简历
- **Excel** — 做数据表、统计成绩、课程表
- **PPT** — 课堂展示、毕业答辩、社团汇报

不会用 Office 的大学生，就像不会写字一样，寸步难行。

## 获取 Office

### 方法一：学校正版化（推荐）

很多学校跟微软有合作，学生可以**免费**使用正版 Office 365。

1. 去学校信息中心/网络中心咨询
2. 用学校邮箱注册 Office 365 教育版
3. 下载安装，登录学校账号即可使用

### 方法二：WPS（免费替代）

如果学校没有正版化，可以用 WPS：
- **官网**：[https://www.wps.cn](https://www.wps.cn)
- 免费、国产、兼容 Office 格式
- 功能基本够用

### 方法三：购买正版

- Office 365 个人版：约 398 元/年
- 一次性购买 Office 2024：约 748 元

::: tip 建议
优先用学校提供的免费正版，其次用 WPS。不建议用破解版，有安全风险。
:::

## Word 基础

### 界面认识

- **标题栏** — 最上方，显示文件名
- **菜单栏** — 开始、插入、页面布局等
- **工具栏** — 各种功能按钮
- **编辑区** — 中间写内容的地方
- **状态栏** — 最下方，显示页码、字数等

### 常用功能

| 功能 | 位置 | 快捷键 |
|------|------|--------|
| 加粗 | 开始 → 加粗 | Ctrl + B |
| 斜体 | 开始 → 斜体 | Ctrl + I |
| 下划线 | 开始 → 下划线 | Ctrl + U |
| 对齐 | 开始 → 段落 | Ctrl + L/E/R |
| 查找替换 | 开始 → 查找 | Ctrl + H |
| 插入图片 | 插入 → 图片 | — |
| 插入表格 | 插入 → 表格 | — |
| 页眉页脚 | 插入 → 页眉页脚 | — |

### 写论文必备技能

#### 1. 标题样式
- 用「标题 1」「标题 2」等样式，不要手动加粗放大
- 这样可以自动生成目录

#### 2. 自动生成目录
- 引用 → 目录 → 自动目录
- 修改内容后右键目录 → 更新域

#### 3. 页码设置
- 插入 → 页码
- 摘要用罗马数字，正文用阿拉伯数字

#### 4. 参考文献
- 引用 → 插入尾注
- 或者用 EndNote/Zotero 管理参考文献

## Excel 基础

### 界面认识

- **单元格** — 每个小格子，用列字母+行数字表示（如 A1、B2）
- **工作表** — 底部的标签页，可以有多个
- **公式栏** — 显示当前单元格的内容或公式

### 常用函数

| 函数 | 用途 | 示例 |
|------|------|------|
| `SUM` | 求和 | `=SUM(A1:A10)` |
| `AVERAGE` | 平均值 | `=AVERAGE(A1:A10)` |
| `MAX` | 最大值 | `=MAX(A1:A10)` |
| `MIN` | 最小值 | `=MIN(A1:A10)` |
| `COUNT` | 计数 | `=COUNT(A1:A10)` |
| `IF` | 条件判断 | `=IF(A1>60,"及格","不及格")` |
| `VLOOKUP` | 查找匹配 | `=VLOOKUP(A1,Sheet2!A:B,2,0)` |

### 常用操作

| 操作 | 方法 |
|------|------|
| 快速求和 | 选中数据 → 右下角自动求和 |
| 筛选 | 数据 → 筛选 |
| 排序 | 数据 → 排序 |
| 条件格式 | 开始 → 条件格式 |
| 数据透视表 | 插入 → 数据透视表 |

::: tip Excel 快捷键
- `Ctrl + C/V` — 复制/粘贴
- `Ctrl + Z` — 撤销
- `Ctrl + S` — 保存
- `F2` — 编辑单元格
- `Ctrl + 方向键` — 跳转到数据边缘
:::

## PPT 基础

### 界面认识

- **幻灯片列表** — 左侧显示所有幻灯片
- **编辑区** — 中间编辑当前幻灯片
- **设计** — 右侧可以设置主题、变体

### 做好 PPT 的原则

#### 1. 简洁
- 每页不要放太多文字
- 用关键词，不要写整段话
- 字号至少 24pt

#### 2. 统一
- 全部用同一个主题/配色
- 字体统一（中文用微软雅黑，英文用 Arial）
- 对齐方式统一

#### 3. 视觉化
- 能用图就不用文字
- 用图表代替数据表
- 适当使用动画（但不要太多）

### 常用功能

| 功能 | 位置 |
|------|------|
| 新建幻灯片 | 开始 → 新建幻灯片 |
| 更改版式 | 开始 → 版式 |
| 插入图片 | 插入 → 图片 |
| 插入图表 | 插入 → 图表 |
| 插入形状 | 插入 → 形状 |
| 动画效果 | 动画 → 添加动画 |
| 切换效果 | 切换 → 选择效果 |
| 幻灯片放映 | F5（从头开始）|

## 新手常见问题

### Q：Word 文档怎么保存？
- `Ctrl + S` 随时保存
- 第一次保存会弹出对话框，让你选择保存位置和文件名
- 建议保存到 D 盘，不要存桌面

### Q：Excel 里输入数字变成日期了怎么办？
- 这是 Excel 自动识别导致的
- 输入前先输入一个英文单引号 `'`，再输入数字
- 或者右键单元格 → 设置单元格格式 → 选「文本」

### Q：PPT 做得好丑怎么办？
- 用模板！新建 PPT 时搜"模板"，选一个好看的
- 保持简洁，不要放太多字
- 用统一的字体和配色

### Q：文件打不开，提示"文件已损坏"？
- 可能是文件下载不完整，重新下载试试
- 或者文件格式不支持，检查一下扩展名
- 用 WPS 打开试试，兼容性更好
