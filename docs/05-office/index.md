# Office 办公软件

## 为什么需要学 Office？

大学四年你几乎每天都要用到 Office：
- **Word** — 写论文、写报告、写简历
- **Excel** — 做数据表、统计成绩、课程表
- **PPT** — 课堂展示、毕业答辩、社团汇报

---

## 获取 Office

### 方法一：学校正版化（推荐）

很多学校跟微软有合作，学生可以**免费**使用正版 Office 365。

1. 去学校信息中心/网络中心咨询
2. 用学校邮箱注册 Office 365 教育版
3. 下载安装，登录学校账号即可使用

### 方法二：WPS（免费替代）

如果学校没有正版化，可以用 WPS：
- 官网：wps.cn
- 免费、国产、兼容 Office 格式

> 💡 优先用学校提供的免费正版，其次用 WPS。不建议用破解版，有安全风险。

## Word 基础

### 常用功能

| 功能 | 快捷键 |
|------|--------|
| 加粗 | Ctrl + B |
| 斜体 | Ctrl + I |
| 下划线 | Ctrl + U |
| 查找替换 | Ctrl + H |
| 保存 | Ctrl + S |

### 写论文必备技能

**1. 标题样式**
- 用「标题 1」「标题 2」等样式，不要手动加粗放大
- 这样可以自动生成目录

**2. 自动生成目录**
- 引用 → 目录 → 自动目录
- 修改内容后右键目录 → 更新域

**3. 页码设置**
- 插入 → 页码
- 摘要用罗马数字，正文用阿拉伯数字

## PPT 基础

### 做好 PPT 的原则

**1. 简洁**
- 每页不要放太多文字
- 用关键词，不要写整段话
- 字号至少 24pt

**2. 统一**
- 全部用同一个主题/配色
- 字体统一（中文用微软雅黑，英文用 Arial）

**3. 视觉化**
- 能用图就不用文字
- 用图表代替数据表

### PPT 模板

不想从零开始做？可以用模板：
- **Office 官方模板** — 新建时搜索
- **WPS 模板** — 内置很多免费模板
- **Canva** — canva.cn 在线设计，模板丰富

## 新手常见问题

### Q：Word 文档怎么保存？
- Ctrl + S 随时保存
- 第一次保存会弹出对话框，让你选择保存位置和文件名
- 建议保存到 D 盘，不要存桌面

### Q：Excel 里输入数字变成日期了怎么办？
- 输入前先输入一个英文单引号 `'`，再输入数字
- 或者右键单元格 → 设置单元格格式 → 选「文本」

### Q：PPT 做得好丑怎么办？
- 用模板！新建 PPT 时搜"模板"，选一个好看的
- 保持简洁，不要放太多字
- 用统一的字体和配色

### Q：文件打不开，提示"文件已损坏"？
- 可能是文件下载不完整，重新下载试试
- 用 WPS 打开试试，兼容性更好
