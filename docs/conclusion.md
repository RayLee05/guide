## 遇到问题怎么办？

### 1. 搜索

- [百度](https://baidu.com) — 国内搜索首选
- [Google](https://google.com) — 需翻墙，质量更高

### 2. 看视频

- [B 站](https://bilibili.com) — 搜"电脑入门"、"Steam 教程"、"GitHub 入门"

### 3. 问 AI

- [DeepSeek](https://chat.deepseek.com) — 免费，国内直接用
- [豆包](https://doubao.com) — 免费，国内直接用

**怎么问？**
- "怎么把文件整理到 D 盘？"
- "Excel 怎么用 VLOOKUP？"
- "这段代码报错了，帮我看看：（贴代码）"

### 4. 问人

- 室友、学长学姐、班级群

## 新手避坑

| 坑 | 怎么避免 |
|----|---------|
| 文件全堆桌面 | 建文件夹结构 |
| 所有账号同密码 | 重要账号用不同密码 |
| 下载软件不看来源 | 只从官网下载 |
| 不备份文件 | 存两个地方 |
| 不保存就关闭 | 养成 Ctrl+S 习惯 |
