## 遇到问题怎么办？

### 1. 搜索

大部分问题，搜一下就能找到答案：

- **百度** — baidu.com 国内搜索首选
- **Google** — google.com 需要翻墙，但搜索质量更高
- **Bing** — bing.com 微软搜索引擎，国内可用

### 2. 看视频教程

文字看不懂？看视频更直观：

- **B 站** — bilibili.com 最好的学习平台！搜"电脑入门"、"Steam 教程"、"GitHub 入门"等关键词，有大量免费教程
- **YouTube** — youtube.com 需要翻墙，但教程质量很高

### 3. 问 AI

遇到具体问题，直接问 AI 最快：

- **DeepSeek** — chat.deepseek.com 免费，国内直接用
- **豆包** — doubao.com 免费，国内直接用
- **Kimi** — kimi.moonshot.cn 免费，长文本处理强
- **通义千问** — tongyi.aliyun.com 免费，功能全面

**怎么问 AI？**

把你的问题详细描述给 AI，越详细越好。举几个例子：

- "怎么把下载的文件整理到 D 盘的文件夹里，能教我吗？"
- "Excel 里怎么用 VLOOKUP 函数匹配两个表格的数据？"
- "PPT 怎么设置动画让文字一个一个出现？"
- "Git 提交代码时报错 'permission denied'，怎么解决？"
- "Steam 下载游戏特别慢，有什么解决办法？"
- "帮我用 Python 写一个批量重命名文件的脚本"
- "这段代码报错了，帮我看看哪里有问题：（贴上代码）"

### 4. 问同学/学长

身边的人往往能给你最直接的帮助：
- **室友** — 一起研究，互相帮助
- **学长学姐** — 他们踩过的坑就是你的经验
- **班级群** — 在群里问一句，可能很多人帮你

### 5. 官方文档/帮助

软件自带的帮助文档也是很好的资源：
- 软件里的「帮助」菜单
- 官方网站的「FAQ」或「常见问题」

---

## 学习建议

1. **不要怕犯错** — 电脑搞不坏的，大胆尝试
2. **边学边用** — 看完教程马上动手操作
3. **养成好习惯** — 文件分类、定期备份、密码管理
4. **保持好奇心** — 新工具、新技术不断出现，持续学习

---

## 新手最容易踩的坑

| 坑 | 怎么避免 |
|----|---------|
| 文件全堆桌面 | 建立文件夹结构，定期整理 |
| 所有账号用同一个密码 | 重要账号用不同密码 |
| 下载软件不看来源 | 只从官网下载，不要点广告链接 |
| 随便点邮件里的链接 | 先确认发件人是否可信 |
| 不备份重要文件 | 重要文件至少存两个地方（电脑+网盘/U盘） |
| 不保存就关闭文档 | 养成 Ctrl + S 的习惯 |
