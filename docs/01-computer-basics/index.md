# 电脑基础

## 电脑是什么？

电脑是一种可以帮你处理各种事情的工具：
- **写文档** — 论文、报告、简历
- **上网** — 浏览网页、看视频、聊天
- **学习** — 看网课、查资料、做作业
- **娱乐** — 看电影、听音乐、玩游戏

| **单击** | 左键点一下 | 选中文件、点击按钮 |

## 文件管理

### 为什么文件管理很重要？

养成好的文件管理习惯，能让你：
- 3 秒内找到任何文件
- 电脑运行更流畅（磁盘不塞满）
- 重装系统时不会丢失重要资料

### 文件夹结构

建议在 D 盘建一个这样的目录结构：

```
D:/
├── 学习资料/
│   ├── 大一上/
│   └── 大一下/
├── 软件安装包/
├── 项目文件/
├── 个人资料/
└── 临时文件/
```

> 💡 不要把文件全堆在桌面！桌面文件太多会拖慢系统。不要把重要文件放 C 盘！重装系统时 C 盘会被清空。

### 文件扩展名

| 扩展名 | 类型 | 用什么打开 |
|--------|------|-----------|
| `.docx` | Word 文档 | Microsoft Word / WPS |
| `.xlsx` | Excel 表格 | Microsoft Excel / WPS |
| `.pptx` | PPT 演示 | Microsoft PowerPoint / WPS |
| `.pdf` | PDF 文档 | 浏览器 / Adobe Reader |
| `.zip` / `.rar` | 压缩包 | 7-Zip / WinRAR |
| `.jpg` / `.png` | 图片 | 照片查看器 |
| `.mp4` | 视频 | PotPlayer / VLC |
| `.exe` | 可执行程序 | 直接运行（注意安全！） |

> ⚠️ Windows 默认隐藏文件扩展名。建议开启：文件资源管理器 → 查看 → 勾选「文件扩展名」。

### 回收站

- **恢复文件**：打开回收站 → 右键文件 → 还原
- **彻底删除**：回收站里右键 → 删除
- **清空回收站**：右键回收站图标 → 清空回收站

> 🚨 从 U 盘删除的文件不会进回收站，直接就没了！

### 磁盘清理

用久了电脑会变慢，定期清理：
1. 右键 C 盘 → 属性 → 磁盘清理
2. 设置 → 系统 → 存储 → 开启「存储感知」

## 电脑选购指南

### 先想清楚你需要什么样的电脑？

1. **你的专业是什么？** 文科/商科对电脑要求不高，工科/设计/影视类可能需要高性能
2. **你玩不玩游戏？** 玩大型游戏需要游戏本，不玩游戏轻薄本就够了
3. **你的预算多少？** 4000-6000 是主流区间

### Windows vs Mac

| 对比项 | Windows | Mac |
|--------|---------|-----|
| 价格 | 3000-10000+ | 8000-20000+ |
| 软件兼容性 | 几乎所有软件都支持 | 部分专业软件不支持 |
| 游戏 | 支持大部分游戏 | 游戏很少 |
| 系统稳定性 | 一般 | 很好 |

> 💡 计算机/工科专业选 Windows，设计/影视专业可选 Mac，玩游戏必须 Windows。

### 轻薄本 vs 游戏本

| | 轻薄本 | 游戏本 |
|--|--------|--------|
| 重量 | 1-1.5kg | 2-2.5kg |
| 续航 | 长 | 短 |
| 性能 | 一般 | 强 |
| 适合 | 文科/商科/日常 | 工科/设计/游戏 |
| 价位 | 4000-7000 | 5000-10000+ |

### 关键参数

**CPU（处理器）** — 电脑的"大脑"
- Intel：i5 够用，i7 更强，i9 顶级
- AMD：R5 够用，R7 更强，R9 顶级
- Apple：M 系列芯片

**内存（RAM）** — 决定能同时开多少程序
- 8GB：勉强够用
- 16GB：推荐！日常完全够
- 32GB：重度使用（剪视频、跑大模型）

**硬盘** — 存储文件的地方
- 一定要选 SSD（固态硬盘），不要 HDD
- 容量：512GB 够用，1TB 更舒服

**屏幕**
- 分辨率：1080P 够用，2K 更清晰
- 色域：100% sRGB 起步

### 购买渠道

- **京东自营** — 正品保障、售后方便
- **天猫官方旗舰店** — 正品保障、活动价低
- **拼多多百亿补贴** — 价格最低，认准官方店铺

## 新手常见问题

### Q：电脑卡了怎么办？
- 先等一等，可能只是暂时卡顿
- 按 `Ctrl + Alt + Delete`，选择「任务管理器」，结束无响应的程序
- 完全没反应就长按电源键强制关机

### Q：怎么安装软件？
- 去官网下载安装包（`.exe` 文件）
- 双击运行，一路点「下一步」
- 注意取消勾选捆绑软件

### Q：怎么卸载软件？
- 设置 → 应用 → 已安装的应用 → 找到软件 → 点「卸载」

### Q：电脑中毒了怎么办？
- 大部分"中毒"其实是弹窗广告
- Windows 自带的杀毒软件就够了，全盘扫描
