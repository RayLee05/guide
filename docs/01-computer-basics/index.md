# 电脑基础

欢迎来到第一章！如果你之前没怎么用过电脑，或者只会基本的上网聊天，那这一章就是为你准备的。

## 电脑是什么？

电脑是一种可以帮你处理各种事情的工具：
- **写文档** — 论文、报告、简历
- **上网** — 浏览网页、看视频、聊天
- **学习** — 看网课、查资料、做作业
- **娱乐** — 看电影、听音乐、玩游戏

## 电脑的基本组成

### 硬件（你能看到摸到的部分）

| 部件 | 作用 | 说明 |
|------|------|------|
| **主机** | 电脑的核心 | 台式机是大箱子，笔记本是一体的 |
| **显示器** | 显示画面 | 台式机需要单独买，笔记本自带 |
| **键盘** | 打字输入 | 笔记本自带，台式机需要单独买 |
| **鼠标** | 点击操作 | 笔记本有触控板，但鼠标更方便 |

### 软件（你安装的程序）

| 类型 | 举例 | 用途 |
|------|------|------|
| **操作系统** | Windows、macOS | 电脑运行的基础 |
| **浏览器** | Chrome、Edge | 上网用 |
| **办公软件** | Word、Excel、PPT | 写文档、做表格、做演示 |
| **聊天软件** | 微信、QQ | 跟人交流 |
| **播放器** | PotPlayer、VLC | 看视频、听音乐 |

## 操作系统

### Windows
- **最常见** — 大部分电脑都用 Windows
- **易上手** — 界面简单，容易学
- **软件多** — 几乎所有软件都支持

### macOS
- **苹果电脑专用** — MacBook 用的系统
- **流畅稳定** — 不容易卡顿、中毒
- **适合设计** — 很多设计师用 Mac

::: tip 建议
如果你是新手，建议先用 Windows，等熟悉了再考虑其他系统。
:::

## 鼠标操作

| 操作 | 方法 | 用途 |
|------|------|------|
| **单击** | 左键点一下 | 选中文件、点击按钮 |
| **双击** | 左键快速点两下 | 打开文件、程序 |
| **右键** | 右键点一下 | 弹出菜单 |
| **拖拽** | 按住左键移动 | 移动文件、选择区域 |
| **滚轮** | 滚动中间滚轮 | 上下翻页 |

## 键盘常用键

| 键 | 作用 |
|----|------|
| **Enter** | 回车，确认/换行 |
| **Backspace** | 退格，删除左边字符 |
| **Delete** | 删除，删除右边字符/文件 |
| **Tab** | 跳格，对齐/切换 |
| **Shift** | 上档，输入大写/特殊字符 |
| **Ctrl** | 控制，配合其他键使用 |
| **Alt** | 菜单键，配合其他键使用 |
| **Esc** | 退出，关闭当前操作 |
| **Space** | 空格，输入空格 |

## 常用快捷键

掌握这些快捷键，效率直接翻倍：

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + C` | 复制 |
| `Ctrl + V` | 粘贴 |
| `Ctrl + X` | 剪切 |
| `Ctrl + Z` | 撤销（后悔药！） |
| `Ctrl + S` | 保存 |
| `Ctrl + A` | 全选 |
| `Ctrl + F` | 查找 |
| `Alt + Tab` | 切换窗口 |
| `Win + E` | 打开文件管理器 |
| `Win + D` | 显示桌面 |
| `Win + L` | 锁屏 |

::: tip 小技巧
- `Win` 键就是键盘上那个带 Windows 图标的键，一般在左下角
- 快捷键不需要死记硬背，用多了自然就记住了
- 如果按错了，`Ctrl + Z` 可以撤销大部分操作
:::

## 新手常见问题

### Q：电脑卡了怎么办？
- 先等一等，可能只是暂时卡顿
- 如果鼠标能动但点不了，按 `Ctrl + Alt + Delete`，选择「任务管理器」，结束无响应的程序
- 如果完全没反应，长按电源键强制关机，再重新开机

### Q：怎么安装软件？
- 去官网下载安装包（`.exe` 文件）
- 双击运行，一路点「下一步」
- 注意取消勾选捆绑软件（很多免费软件会偷偷装别的东西）

### Q：怎么卸载软件？
- 设置 → 应用 → 已安装的应用 → 找到要卸载的软件 → 点「卸载」
- 不要直接删除桌面快捷方式，那样软件还在

### Q：电脑中毒了怎么办？
- 不要慌，大部分"中毒"其实是弹窗广告
- 安装杀毒软件（Windows 自带的就够了）全盘扫描
- 如果真的中毒严重，可以重装系统（找懂电脑的同学帮忙）
