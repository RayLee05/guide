# 电脑基础

## 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + C/V/X` | 复制/粘贴/剪切 |
| `Ctrl + Z` | 撤销 |
| `Ctrl + S` | 保存 |
| `Ctrl + A` | 全选 |
| `Alt + Tab` | 切换窗口 |
| `Win + E` | 打开文件管理器 |
| `Win + D` | 显示桌面 |
| `Win + L` | 锁屏 |

## 文件管理

### 文件夹结构（建议放 D 盘）

```
D:/
├── 学习资料/大一上/
├── 软件安装包/
├── 个人资料/
└── 临时文件/
```

### 常见文件类型

| 扩展名 | 打开方式 |
|--------|---------|
| `.docx` | Word / WPS |
| `.xlsx` | Excel / WPS |
| `.pptx` | PowerPoint / WPS |
| `.pdf` | 浏览器 |
| `.zip/.rar` | 7-Zip / WinRAR |
| `.mp4` | PotPlayer / VLC |

> ⚠️ 文件资源管理器 → 查看 → 勾选「文件扩展名」

## 新手问题

| 问题 | 解决方法 |
|------|---------|
| 电脑卡了 | Ctrl+Alt+Delete → 任务管理器 → 结束无响应程序 |
| 怎么安装软件 | 去官网下载 .exe → 双击运行 → 取消勾选捆绑软件 |
| 怎么卸载软件 | 设置 → 应用 → 已安装的应用 → 卸载 |
| 电脑中毒 | Windows 自带杀毒全盘扫描 |
