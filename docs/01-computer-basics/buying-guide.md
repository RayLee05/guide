# 电脑选购指南

## 先想清楚一个问题：你需要什么样的电脑？

买电脑之前，先问自己几个问题：

1. **你的专业是什么？** 文科/商科对电脑要求不高，工科/设计/影视类可能需要高性能
2. **你玩不玩游戏？** 玩大型游戏需要游戏本，不玩游戏轻薄本就够了
3. **你的预算多少？** 4000-6000 是主流区间

## Windows vs Mac

| 对比项 | Windows | Mac |
|--------|---------|-----|
| 价格 | 3000-10000+ | 8000-20000+ |
| 软件兼容性 | 几乎所有软件都支持 | 部分专业软件不支持 |
| 游戏 | 支持大部分游戏 | 游戏很少 |
| 系统稳定性 | 一般（需要自己维护） | 很好（基本不用管） |
| 生态 | 开放 | 封闭但体验好 |

::: tip 建议
- 如果你是**计算机/工科专业**：Windows 更方便，很多专业软件只有 Windows 版
- 如果你是**设计/影视专业**：Mac 是不错的选择
- 如果你**玩游戏**：必须 Windows
- 如果你**预算有限**：Windows 性价比更高
:::

## 轻薄本 vs 游戏本

### 轻薄本
- **优点**：轻便（1-1.5kg）、续航长、颜值高
- **缺点**：性能一般，玩不了大型游戏
- **适合**：文科/商科/日常办公/不玩游戏的同学
- **价位**：4000-7000

### 游戏本
- **优点**：性能强、能玩大型游戏、能跑专业软件
- **缺点**：重（2-2.5kg）、续航短、风扇声音大
- **适合**：工科/设计/玩游戏的同学
- **价位**：5000-10000+

## 关键参数怎么看

### CPU（处理器）
电脑的"大脑"，决定了运行速度。

- **Intel**：i5 够用，i7 更强，i9 顶级
- **AMD**：R5 够用，R7 更强，R9 顶级
- **Apple**：M 系列芯片

::: tip 看型号数字
比如 Intel i7-13700H：
- i7 = 系列（越高越强）
- 13 = 第 13 代（越大越新）
- 700 = 具体型号（越大越强）
- H = 标压（性能强），U = 低压（省电）
:::

### 内存（RAM）
决定了你能同时开多少个程序。

- **8GB**：勉强够用，开多了会卡
- **16GB**：推荐！日常使用完全够
- **32GB**：重度使用（剪视频、跑大模型）

### 硬盘
存储文件的地方。

- **SSD（固态硬盘）**：速度快，必选！
- **HDD（机械硬盘）**：速度慢，不推荐
- **容量**：512GB 够用，1TB 更舒服

::: warning 注意
一定要选 SSD！HDD 会让你的电脑开机要 2 分钟，打开软件转圈圈。
:::

### 屏幕
你每天盯着看的东西，很重要！

- **分辨率**：1080P 够用，2K 更清晰
- **色域**：100% sRGB 起步，设计类需要 P3 色域
- **刷新率**：60Hz 够用，游戏本建议 120Hz+
- **亮度**：300nit 以上，户外也能看清

## 购买渠道

- **京东自营** — 正品保障、售后方便、经常有活动
- **天猫官方旗舰店** — 正品保障、活动价低
- **拼多多百亿补贴** — 价格最低，但要认准官方店铺

::: danger 避坑
- **不要去电脑城买！** 各种套路防不胜防
- **不要听销售推荐！** 他们只会推利润高的
- **买之前先去 B 站看评测！** 搜"XX 笔记本评测"，看真实用户反馈
:::

## 选购流程总结

1. 确定预算
2. 确定需求（轻薄本还是游戏本）
3. B 站搜评测，看真实使用体验
4. 对比几款候选机型的参数
5. 选靠谱渠道购买
