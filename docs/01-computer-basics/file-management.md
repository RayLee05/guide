# 文件管理

## 为什么文件管理很重要？

你有没有见过那种桌面全是文件、下载文件夹里有几百个文件、找什么都找不到的电脑？没错，那就是没有文件管理意识的后果。

养成好的文件管理习惯，能让你：
- 3 秒内找到任何文件
- 电脑运行更流畅（磁盘不塞满）
- 重装系统时不会丢失重要资料

## 文件夹结构

建议在 C 盘（系统盘）之外的盘（比如 D 盘）建一个这样的目录结构：

```
D:/
├── 学习资料/
│   ├── 大一上/
│   │   ├── 高等数学/
│   │   ├── 大学英语/
│   │   └── C语言程序设计/
│   └── 大一下/
├── 软件安装包/
├── 项目文件/
├── 个人资料/
└── 临时文件/
```

::: tip 小技巧
- **不要把文件全堆在桌面！** 桌面文件太多会拖慢系统
- **不要把重要文件放 C 盘！** 重装系统时 C 盘会被清空
- **定期清理下载文件夹**，那里是垃圾文件的重灾区
:::

## 文件扩展名

文件扩展名就是文件名最后那个 `.xxx`，它告诉你这是什么类型的文件。

### 常见文件类型

| 扩展名 | 类型 | 用什么打开 |
|--------|------|-----------|
| `.docx` | Word 文档 | Microsoft Word / WPS |
| `.xlsx` | Excel 表格 | Microsoft Excel / WPS |
| `.pptx` | PPT 演示 | Microsoft PowerPoint / WPS |
| `.pdf` | PDF 文档 | 浏览器 / Adobe Reader |
| `.zip` / `.rar` | 压缩包 | 7-Zip / WinRAR |
| `.jpg` / `.png` | 图片 | 照片查看器 |
| `.mp4` | 视频 | PotPlayer / VLC |
| `.exe` | 可执行程序 | 直接运行（注意安全！） |
| `.txt` | 纯文本 | 记事本 |

::: warning 注意
Windows 默认隐藏文件扩展名。建议开启显示：
1. 打开「文件资源管理器」
2. 点击顶部「查看」
3. 勾选「文件扩展名」

这样你就能看到 `.exe`、`.bat` 等危险文件的真实类型，避免被伪装文件骗到。
:::

## 回收站

删掉的文件不会立刻消失，它们会先待在「回收站」里。

- **恢复文件**：打开回收站 → 右键文件 → 还原
- **彻底删除**：回收站里右键 → 删除（这个就真的没了）
- **清空回收站**：右键回收站图标 → 清空回收站

::: danger 注意
从 U 盘或移动硬盘删除的文件**不会进回收站**，直接就没了！删之前一定要确认清楚。
:::

## 磁盘清理

用久了电脑会变慢，很大原因是磁盘里堆满了垃圾文件。

### 方法一：Windows 自带清理
1. 右键 C 盘 → 属性 → 磁盘清理
2. 勾选要清理的内容（临时文件、缩略图缓存等）
3. 点击「清理系统文件」可以清理更多

### 方法二：存储感知（推荐开启）
1. 设置 → 系统 → 存储
2. 开启「存储感知」
3. 设置自动清理频率（比如每周自动清理临时文件）
