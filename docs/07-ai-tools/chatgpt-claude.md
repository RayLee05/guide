# ChatGPT / Claude

## 为什么放在一起说？

ChatGPT（OpenAI 出品）和 Claude（Anthropic 出品）是目前全球最强的两个 AI 对话工具。但因为它们都需要翻墙才能访问，而且注册门槛较高（需要国外手机号），所以这里只做简要介绍。

## Claude（推荐）

Anthropic 公司出品，目前公认的代码和写作能力最强的 AI。

- **官网**：[https://claude.ai](https://claude.ai)
- **最新模型**：Claude 4.X 系列
  - **Opus 4.7** — 最强旗舰，复杂推理和代码能力顶级
  - **Sonnet 4.6** — 性价比之选，速度快、能力强
  - **Haiku 4.5** — 轻量快速，适合简单任务
- **免费版**：每天有使用次数限制
- **Pro 版**：$20/月，不限次数

### Claude Code

Claude 还有一个很厉害的功能叫 **Claude Code**，是一个命令行编程助手：

- 直接在终端里跟 Claude 对话
- 让 Claude 帮你改代码、运行命令
- 支持 CLI、桌面应用、VS Code / JetBrains 插件

```bash
# 安装
npm install -g @anthropic-ai/claude-code

# 使用
claude
```

## ChatGPT

OpenAI 公司出品，全球用户最多的 AI。

- **官网**：[https://chat.openai.com](https://chat.openai.com)
- **最新模型**：GPT-5 / o3 系列（o3 擅长推理和数学）
- **免费版**：有次数限制
- **Plus 版**：$20/月
