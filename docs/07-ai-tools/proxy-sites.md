# 反代网站

## 什么是反代网站？

反代网站（反向代理网站）是指第三方搭建的网站，通过技术手段接入 ChatGPT、Claude 等 AI 的接口，让你**不需要翻墙、不需要注册国外账号**就能直接使用这些 AI。

简单说：别人帮你翻墙、帮你注册，你直接用就行。

## 费用情况

反代网站**不是免费的**，而且价格不便宜：

- 大部分反代按月收费，一般 **30-100 元/月** 不等
- 有些按 token（使用量）计费，用得多花得多
- 免费反代很少，而且通常限制很多（排队、限次数、限模型）

::: warning 注意
如果你看到"免费无限用"的反代，要特别警惕 — 可能是钓鱼网站，或者会偷偷收集你的数据。
:::

## 怎么找反代网站？

反代网站变化很快，今天能用明天可能就挂。以下方法帮你找到最新的：

### 方法一：GitHub 搜索

在 GitHub 搜以下关键词，很多开源项目会附带公开可用的地址：
- `chatgpt web`
- `claude web`
- `one-api`（很多反代用这个项目搭建）

### 方法二：搜索引擎

搜关键词：
- `ChatGPT 反代 2026`
- `Claude 镜像站`
- `AI 反代网站`

### 方法三：技术社区

- **V2EX** — 技术社区，经常有人分享
- **掘金** — 开发者社区
- **Telegram 群组** — 搜 AI 相关群组
- **Twitter/X** — 搜中文关键词

### 方法四：朋友推荐

最靠谱的方式 — 问已经在用的同学或朋友。

## 使用反代的注意事项

### ⚠️ 安全提醒

- **不要输入敏感信息** — 反代网站的运营者理论上能看到你发的内容
- **不要用来处理机密文件** — 比如公司内部文档、身份证照片等
- **注意识别钓鱼网站** — 有些假的反代网站会盗号

### ⚠️ 使用建议

- **多个备选** — 反代网站经常挂，多收藏几个
- **重要的活用官方** — 如果任务很重要，还是翻墙用官方版
- **及时保存** — 反代网站可能随时关闭，重要内容及时保存

## 反代 vs 国内 AI

| | 反代（ChatGPT/Claude） | 国内 AI（DeepSeek 等） |
|--|----------------------|----------------------|
| 能力 | 更强 | 接近 |
| 稳定性 | 一般 | 稳定 |
| 价格 | 收费（30-100 元/月） | 免费 |
| 门槛 | 需要找网站、付费 | 直接注册 |
| 推荐 | 重要任务 | 日常使用 |

::: tip 我的建议
- **日常使用**：国内 AI（免费、稳定）
- **重要任务**：反代或翻墙用官方 Claude
- **预算有限**：先用国内 AI，够用就不用花反代的钱
:::
