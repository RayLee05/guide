## 怎么问 AI

❌ `帮我写个程序`

✅ `帮我用 Python 写一个程序，功能是读取 CSV 文件，统计每列平均值，输出到新文件。用 pandas 库。`

**公式：角色 + 任务 + 要求 + 格式**

## 反代网站

反代 = 第三方接入 ChatGPT/Claude，不需要翻墙直接用

**费用**：30-100 元/月

**怎么找**：
- GitHub 搜 `chatgpt web`、`claude web`、`one-api`
- 搜 `ChatGPT 反代 2026`
- 问朋友推荐

**反代 vs 国内 AI**

| | 反代 | 国内 AI |
|--|------|---------|
| 能力 | 更强 | 接近 |
| 价格 | 30-100 元/月 | 免费 |
| 稳定性 | 一般 | 稳定 |

## 注意事项

✅ 可以：学习、写代码、翻译、润色

❌ 不要：
- 让 AI 直接写论文交上去
- 发密码、身份证号等敏感信息
- 完全相信 AI（有时会编造）
