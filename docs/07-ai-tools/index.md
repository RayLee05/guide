# AI 工具

## 什么是 AI？

AI（人工智能）是一种可以像人一样"思考"的计算机程序。你跟它说话（打字），它会回答你。

现在最火的 AI 是**对话式 AI**，就像一个什么都懂的聊天对象：
- 你问它问题，它回答
- 你让它写东西，它帮你写
- 你给它代码，它帮你改

---

## 为什么大学生必须会用 AI？

2026 年了，AI 已经不是什么新鲜事。它能帮你：

- **写代码** — 告诉它你要什么功能，它帮你写
- **写论文** — 帮你润色、翻译、查资料
- **学知识** — 有什么不懂的直接问，比搜索引擎好用
- **做 PPT** — 帮你生成大纲和内容
- **解决 bug** — 把报错信息丢给它，它帮你分析

## AI 能做什么？

### AI 擅长的事

| 场景 | 例子 |
|------|------|
| 写代码 | "帮我用 Python 写一个贪吃蛇" |
| 翻译 | "把这段英文翻译成中文" |
| 解释概念 | "用通俗的语言解释什么是量子计算" |
| 写文案 | "帮我写一封求职邮件" |
| Debug | "这段代码报错了，帮我看看" |
| 总结 | "帮我总结这篇论文的要点" |

### AI 不擅长的事

| 场景 | 原因 |
|------|------|
| 最新新闻 | AI 的知识有截止日期 |
| 精确数据 | AI 有时会编造数据 |
| 专业判断 | AI 不是医生/律师，不能替代专业人士 |

## 怎么跟 AI 说话？

### 好的提问方式

❌ 不好的提问：`帮我写个程序`

✅ 好的提问：`帮我用 Python 写一个程序，功能是读取一个 CSV 文件，统计每列的平均值，然后输出结果到新的 CSV 文件。用 pandas 库。`

### 提问公式

**角色 + 任务 + 要求 + 格式**

例如：`你是一个 Python 专家（角色），帮我写一个爬虫程序（任务），要求能处理反爬（要求），代码要有注释（格式）`

## 反代网站

### 什么是反代网站？

反代网站（反向代理网站）是指第三方搭建的网站，通过技术手段接入 ChatGPT、Claude 等 AI 的接口，让你**不需要翻墙、不需要注册国外账号**就能直接使用这些 AI。

### 费用情况

反代网站**不是免费的**，而且价格不便宜：
- 大部分反代按月收费，一般 **30-100 元/月** 不等
- 有些按 token（使用量）计费

> ⚠️ 如果你看到"免费无限用"的反代，要特别警惕 — 可能是钓鱼网站。

### 怎么找反代网站？

- **GitHub 搜索** — 搜 `chatgpt web`、`claude web`、`one-api`
- **搜索引擎** — 搜 `ChatGPT 反代 2026`、`Claude 镜像站`
- **技术社区** — V2EX、掘金、Telegram 群组
- **朋友推荐** — 最靠谱的方式

### 反代 vs 国内 AI

| | 反代（ChatGPT/Claude） | 国内 AI（DeepSeek 等） |
|--|----------------------|----------------------|
| 能力 | 更强 | 接近 |
| 稳定性 | 一般 | 稳定 |
| 价格 | 收费（30-100 元/月） | 免费 |
| 门槛 | 需要找网站、付费 | 直接注册 |

> 💡 日常使用用国内 AI（免费、稳定），重要任务用反代或翻墙用官方。

### 使用 AI 的注意事项

**可以做的**
- 用 AI 学习新知识
- 让 AI 帮你写代码
- 用 AI 翻译、润色文字

**不要做的**
- 不要让 AI 直接帮你写论文交上去 — 查重会暴露，而且你学不到东西
- 不要把敏感信息发给 AI — 密码、身份证号等
- 不要完全相信 AI — 它有时会编造信息

## 新手常见问题

### Q：AI 回答的内容不准确怎么办？
- AI 有时会"一本正经地胡说八道"
- 重要信息一定要自己验证
- 可以换个方式再问一遍，看看回答是否一致

### Q：怎么让 AI 帮我写代码？
- 用自然语言描述你要什么功能
- 比如："帮我用 Python 写一个计算器，支持加减乘除"
- AI 会给你代码，你可以直接复制运行

### Q：AI 和搜索引擎有什么区别？
- 搜索引擎给你一堆链接，你自己找答案
- AI 直接给你答案，还能解释、讨论
- 搜索引擎适合找最新信息，AI 适合解决具体问题

### Q：哪个 AI 最好用？
- 没有绝对最好的，各有特点
- 建议都试试，找到适合自己的
- 日常用 DeepSeek，重要任务用 Claude/ChatGPT（反代）
