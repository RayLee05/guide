# 游戏平台

大学生活不只有学习，适当放松也很重要！

## 什么是游戏平台？

游戏平台是购买、下载、管理游戏的软件。就像淘宝是买东西的，Steam 是买游戏的。

### 主流游戏平台

| 平台 | 公司 | 特点 |
|------|------|------|
| **Steam** | Valve | 最大、游戏最多、经常打折 |
| **Epic Games** | Epic | 每周送免费游戏 |
| **WeGame** | 腾讯 | 国内游戏为主 |
| **暴雪战网** | 暴雪 | 魔兽、守望先锋等 |

::: tip 建议
新手先用 Steam，游戏最多、社区最好、打折最狠。
:::

## Watt Toolkit

Watt Toolkit（原名 Steam++）是一个免费工具，主要功能：

- **加速 Steam** — 让 Steam 商店加载更快
- **加速 GitHub** — 让 GitHub 访问更快（不用翻墙）
- **本地加速** — 不走代理，安全可靠

## Steam

Steam 是全球最大的 PC 游戏平台，由 Valve 公司运营。你可以把它理解为"游戏界的淘宝"：

- **游戏商店** — 买游戏、等打折
- **游戏库** — 管理你买过的所有游戏
- **社区** — 看评测、交朋友
- **创意工坊** — 下载 MOD（游戏修改内容）

### Steam 能做什么？

| 功能 | 说明 |
|------|------|
| **买游戏** | 几十万款游戏任你选 |
| **打折促销** | 夏促、冬促打折力度很大 |
| **社区交流** | 看评测、发帖讨论 |
| **创意工坊** | 下载 MOD，给游戏加新内容 |
| **远程同乐** | 跟朋友一起玩游戏 |
| **家庭共享** | 跟家人共享游戏库 |

### 游戏类型

| 类型 | 英文 | 举例 |
|------|------|------|
| 动作 | Action | GTA、只狼 |
| 冒险 | Adventure | 塞尔达、空洞骑士 |
| 角色扮演 | RPG | 巫师3、最终幻想 |
| 射击 | FPS/TPS | CS2、Apex |
| 策略 | Strategy | 文明6、全面战争 |
| 模拟 | Simulation | 模拟城市、欧洲卡车 |
| 独立 | Indie | 星露谷、哈迪斯 |
