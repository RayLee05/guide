# Steam 使用教程

## 注册 Steam 账号

### 1. 下载 Steam

👉 官网下载：[https://store.steampowered.com](https://store.steampowered.com)

点击「安装 Steam」，下载安装包，一路下一步安装。

### 2. 创建账号

1. 打开 Steam 客户端
2. 点击「创建新账号」
3. 填写邮箱、密码、验证码
4. 验证邮箱
5. 登录成功！

::: tip 建议
- **用户名要想好** — 以后改名要花钱
- **密码要复杂** — Steam 账号值钱，别被盗
- **开启 Steam 令牌** — 保护账号安全
:::

## 购买游戏

### 浏览游戏

- **商店首页** — 推荐游戏、新品
- **浏览** — 按类别、标签筛选
- **搜索** — 直接搜游戏名

### 购买流程

1. 找到想买的游戏
2. 点击「添加至购物车」
3. 点击「为自己购买」
4. 选择支付方式
5. 完成支付

### 支付方式

| 方式 | 说明 |
|------|------|
| **支付宝** | 最方便，推荐 |
| **微信支付** | 也方便 |
| **银联** | 需要银行卡 |
| **Visa/Mastercard** | 国外信用卡 |

### 游戏打折

Steam 经常有大型打折活动：

| 活动 | 时间 | 折扣力度 |
|------|------|---------|
| **夏促** | 6-7 月 | 最大！ |
| **冬促** | 12-1 月 | 很大 |
| **秋促** | 11 月 | 较大 |
| **春节促** | 1-2 月 | 较大 |

::: tip 省钱技巧
- **加入愿望单** — 打折时会邮件通知你
- **看评测** — 买之前先看「特别好评」还是「多半差评」
- **等打折** — 原价买游戏太亏了，等夏促/冬促
- **买捆绑包** — 多个游戏一起买更便宜
:::

## 退款政策

买了游戏不喜欢？Steam 支持退款！

### 退款条件

- 购买后 **14 天内**
- 游戏时间 **不超过 2 小时**

### 退款流程

1. 打开 Steam → 帮助 → Steam 客户支持
2. 找到要退款的游戏
3. 选择退款原因
4. 提交申请
5. 一般 1-3 天内退款到原支付方式

## Steam 令牌

为了账号安全，强烈建议开启 Steam 令牌：

1. 下载 **Steam 手机 App**
2. 登录你的账号
3. 开启 Steam 令牌（手机验证器）
4. 以后每次登录都需要手机确认

::: warning 为什么必须开？
- 防止账号被盗
- 开启后 15 天才能交易
- 保护你的游戏库
:::

## 好友系统

- **添加好友** — 搜索用户名或发好友链接
- **一起玩游戏** — 很多游戏支持多人联机
- **查看动态** — 看朋友在玩什么

## 常见问题

### Q：Steam 打不开/加载慢？
- 用 [Watt Toolkit](./watt-toolkit.md) 加速
- 或者用翻墙工具

### Q：游戏下载慢？
- 设置 → 下载 → 更换下载地区
- 试试香港、日本、韩国

### Q：买游戏显示「此产品在您所在地区不可用」？
- 有些游戏有地区限制
- 可以让国外的朋友送你（Steam 礼物）

### Q：游戏安装后打不开？
- 检查电脑配置是否达到游戏最低要求
- 更新显卡驱动
- 验证游戏文件完整性：右键游戏 → 属性 → 本地文件 → 验证

### Q：Steam 账号被盗了怎么办？
- 立即修改密码
- 联系 Steam 客服
- 开启 Steam 令牌防止再次被盗
