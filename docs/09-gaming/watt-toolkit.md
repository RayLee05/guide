# Watt Toolkit 使用教程

## Watt Toolkit 是什么？

Watt Toolkit（原名 Steam++）是一个免费、开源的工具箱，主要用来加速 Steam 和 GitHub。

### 为什么需要它？

- **Steam 商店加载慢** — 国内访问 Steam 经常卡
- **GitHub 访问慢** — 下载代码、看文档很慢
- **不想翻墙** — Watt Toolkit 用的是本地加速，不需要代理

### 原理是什么？

Watt Toolkit 通过修改本地 hosts 文件，把 Steam/GitHub 的域名指向更快的 IP 地址。这种方式：

- ✅ 不走代理，速度快
- ✅ 免费，无流量限制
- ✅ 安全，不经过第三方服务器
- ❌ 只能加速特定网站

## 下载安装

### 下载地址

👉 官网：[https://steampp.net](https://steampp.net)

👉 GitHub：[https://github.com/BeyondDimension/SteamTools/releases](https://github.com/BeyondDimension/SteamTools/releases)

### 安装步骤

1. 下载安装包（Windows 选 `.exe`）
2. 双击安装，一路下一步
3. 安装完成后打开

## 使用方法

### 加速 Steam

1. 打开 Watt Toolkit
2. 在首页找到「加速」功能
3. 勾选「Steam」
4. 点击「一键加速」
5. 等待加速完成

加速成功后，Steam 商店会加载得很快！

### 加速 GitHub

1. 同样在「加速」功能里
2. 勾选「GitHub」
3. 点击「一键加速」
4. 现在访问 GitHub 会快很多

::: tip 两个一起加速
Steam 和 GitHub 可以同时加速，互不影响。
:::

## 常见问题

### Q：加速后还是慢？
- 尝试切换加速节点
- 检查是否有其他代理软件冲突
- 重启 Watt Toolkit

### Q：安全吗？
- Watt Toolkit 是开源软件，代码公开透明
- 不收集用户数据
- 只修改本地 hosts 文件，不走代理

### Q：需要一直开着吗？
- 是的，加速需要保持 Watt Toolkit 运行
- 可以最小化到托盘，不占桌面

### Q：跟 Clash 冲突吗？
- 可能会冲突，因为都涉及网络代理
- 建议：用 Clash 翻墙，用 Watt Toolkit 加速 Steam/GitHub
- 如果冲突，可以先关掉 Clash 的系统代理，单独用 Watt Toolkit

### Q：手机上能用吗？
- Watt Toolkit 主要是 PC 端工具
- 手机上访问 Steam/GitHub 可以用翻墙工具
