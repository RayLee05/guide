# 校园网

## 什么是校园网？

校园网是学校提供的网络服务，覆盖宿舍、图书馆、教学楼等区域。每个学生都有自己的校园网账号。

---

## 开通校园网

### 1. 获取账号

- 入学时学校会发给你校园网账号和密码
- 账号通常是学号，密码是身份证后几位或统一初始密码
- 去学校网络中心（信息中心）咨询

### 2. 缴费

- 校园网一般按月或按学期收费
- 价格因学校而异，一般 20-50 元/月
- 去学校网络中心或通过学校 APP 缴费

### 3. 连接方式

**有线连接（宿舍）**
1. 准备一根网线
2. 一头插墙上网络接口，一头插电脑
3. 打开浏览器，会自动跳转到认证页面
4. 输入账号密码登录

**无线连接（WiFi）**
1. 连接学校的 WiFi（名称一般是学校缩写）
2. 打开浏览器，会自动跳转到认证页面
3. 输入账号密码登录

> 💡 常见 WiFi 名称：`XXX大学` 或 `XXXU`、`XXX-Student`

## 常见问题

### Q：连上网了但打不开网页？
- 检查是否登录了认证页面
- 有些学校需要每次连接都重新认证
- 尝试访问任意 HTTP 网站触发认证页面跳转

### Q：网速很慢？
- 高峰期（晚上 8-11 点）大家都在线，网速会慢
- 尝试用有线连接，比 WiFi 稳定
- 去网络中心咨询是否可以升级带宽

### Q：一个账号能连几台设备？
- 因学校而异，一般 2-3 台
- 超出限制需要去网络中心申请

### Q：校园网能翻墙吗？
- 校园网本身不能翻墙
- 需要自己配置 [翻墙工具](../06-vpn-proxy/index.md)

### Q：怎么知道我连上校园网了？
- 连接 WiFi 后，打开浏览器
- 如果自动跳转到登录页面，说明连上了
- 输入账号密码登录后就能上网了

### Q：登录页面没弹出来怎么办？
- 尝试访问任意 HTTP 网站（不是 HTTPS）
- 或者手动打开学校网络中心的登录页面

### Q：网费怎么交？
- 去学校网络中心现场缴费
- 或者通过学校 APP/公众号缴费

### Q：宿舍几个人可以共用一个账号吗？
- 一般不行，一个账号只能同时登录几台设备
- 建议每人办一个账号

## 校园网的用途

- **选课** — 教务系统一般只有校园网内才能访问
- **查成绩** — 同样需要校园网
- **图书馆资源** — 知网、万方等学术资源通过校园网免费下载
- **打印** — 学校打印系统需要校园网

> 💡 学校图书馆的数据库资源很值钱！通过校园网可以免费下载知网、万方等论文，出了学校就要付费。

## 校园网 vs 手机流量

| | 校园网 | 手机流量 |
|--|--------|---------|
| 速度 | 一般较快 | 看信号 |
| 稳定 | 有线很稳定 | 不太稳定 |
| 价格 | 便宜 | 较贵 |
| 覆盖 | 校园内 | 全国 |
| 用途 | 学习、下载 | 日常使用 |

> 💡 在宿舍和图书馆用校园网，外出时用手机流量。手机也可以连校园网 WiFi，省流量。
