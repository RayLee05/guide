# 邮箱

## 为什么需要邮箱？

邮箱在大学里非常重要：
- **学校通知** — 学校、学院、老师发通知基本都用邮箱
- **作业提交** — 有些老师要求用邮箱提交作业
- **注册账号** — 各种网站、软件注册都需要邮箱
- **求职** — 简历上要写邮箱，HR 通过邮箱联系你
- **学术交流** — 联系导师、投稿论文都需要邮箱

## 邮箱类型

| 类型 | 示例 | 用途 |
|------|------|------|
| **学校邮箱** | xxx@xxx.edu.cn | 学校事务、正版软件激活 |
| **Gmail** | xxx@gmail.com | 国际通用、注册国外服务 |
| **Outlook** | xxx@outlook.com | 微软生态、国际通用 |
| **QQ 邮箱** | xxx@qq.com | 国内通用、方便 |
| **163 邮箱** | xxx@163.com | 国内通用 |

::: tip 建议
- **学校邮箱** — 必须有，学校会发
- **Gmail/Outlook** — 建议注册一个，以后用得上
- **QQ 邮箱** — 如果你有 QQ 号就有了，但不建议用于正式场合
:::

## 新手常见问题

### Q：邮箱地址怎么写？
邮箱地址格式：`用户名@域名`
- 用户名是你自己起的，比如 `zhangsan`
- 域名是邮箱服务商的，比如 `qq.com`、`outlook.com`
- 例子：`zhangsan@qq.com`、`zhangsan@outlook.com`

### Q：怎么收邮件？
- 登录邮箱网页版或客户端
- 收件箱里就是别人发给你的邮件
- 未读邮件一般会有标记（加粗或数字）

### Q：怎么发邮件？
- 点「写邮件」或「新建邮件」
- 填写收件人邮箱地址
- 写主题和正文
- 需要发文件就点「添加附件」
- 点「发送」

### Q：邮件发错了能撤回吗？
- 有些邮箱支持撤回（比如 Outlook），但有时间限制
- 大部分情况下发出去就收不回来了
- 所以发送前一定要检查清楚

## 学校邮箱

### 获取方式

- 入学后学校会分配给你一个 edu 邮箱
- 账号一般是学号，密码是统一初始密码
- 去学校信息中心/网络中心咨询

### 使用方式

- **网页版** — 访问学校邮箱登录页面
- **客户端** — 可以用 Outlook、Foxmail 等客户端收发邮件
- **手机** — 手机自带邮件 App 或 Outlook App

### 学校邮箱的好处

- **正版软件** — 很多软件教育版免费（Office 365、GitHub Education 等）
- **学术资源** — 知网、万方等通过学校邮箱认证免费下载
- **教育优惠** — Apple、Adobe 等有教育折扣
- **学术交流** — 用 edu 邮箱更正式、更可信

::: warning 注意
学校邮箱毕业后可能会被注销！重要邮件及时备份，重要账号及时更换邮箱。
:::

## 注册 Gmail

### 为什么要用 Gmail？

- 国际通用，几乎所有国外服务都支持
- 注册 ChatGPT、Claude 等 AI 工具需要
- Google 全家桶（Drive、Docs、Calendar）很方便

### 注册步骤

1. 需要先能访问 Google（需要 [翻墙](/06-vpn-proxy/)）
2. 打开 [accounts.google.com](https://accounts.google.com)
3. 点击「创建账号」
4. 填写姓名、用户名、密码
5. 验证手机号（国内手机号可以）
6. 完成注册

::: tip 提示
- 用户名要起好，以后不好改
- 密码要复杂，开启两步验证
- 记住密码，可以用密码管理器
:::

## 注册 Outlook

### 为什么要用 Outlook？

- 微软官方邮箱，国际通用
- 注册 Microsoft 365、GitHub 等需要
- 不需要翻墙就能注册

### 注册步骤

1. 打开 [outlook.live.com](https://outlook.live.com)
2. 点击「创建免费帐户」
3. 选择邮箱地址（@outlook.com 或 @hotmail.com）
4. 设置密码
5. 填写姓名、生日
6. 验证手机号
7. 完成注册

## 邮件客户端

不想每次开网页收邮件？用邮件客户端：

### Windows
- **Outlook** — 微软官方，功能强大
- **Foxmail** — 腾讯出品，简洁好用
- **Thunderbird** — Mozilla 出品，开源免费

### Mac
- **自带邮件 App** — 够用
- **Outlook for Mac** — 功能更全

### 手机
- **Outlook App** — iOS/Android 都有
- **自带邮件 App** — 够用

### 配置客户端

一般需要以下信息：
- **邮箱地址** — xxx@xxx.com
- **密码** — 你的邮箱密码
- **服务器地址** — 去邮箱官网查（搜"xxx邮箱 POP3 设置"）

## 邮件礼仪

### 邮件格式

```
主题：[课程名称] 关于XX问题请教 - 张三

尊敬的XX老师：

您好！我是XX级XX专业的学生张三。

（正文内容）

祝好！

张三
学号：20240101XX
XX大学XX学院
```

### 注意事项

- **主题要明确** — 让收件人一眼知道邮件内容
- **称呼要得体** — 老师用"尊敬的"，同学用"Hi"
- **正文要简洁** — 说清楚事情，不要废话
- **署名要完整** — 姓名、学号、学院
- **及时回复** — 收到邮件尽量 24 小时内回复
- **检查附件** — 发附件前检查是否添加了

::: warning 常见错误
- 忘记写主题
- 忘记加附件（写了"见附件"但没加）
- 回复所有人（Reply All）发了不该发的内容
- 用 QQ 邮箱给老师发正式邮件（不太正式）
:::
