# 电脑端翻墙配置

## 整体流程

```
下载客户端 → 购买机场订阅 → 导入订阅 → 开启代理 → 上网！
```

## 第一步：下载代理客户端

你需要一个代理客户端软件。目前主流的客户端有好几种，具体用哪个可以问你的机场服务商，他们一般会推荐配套的客户端。

### 下载方式

- **GitHub** — 大部分客户端在 GitHub 上发布，搜关键词就能找到
- **机场官网** — 很多机场会提供下载链接
- **朋友分享** — 让已经翻墙的同学发给你

::: tip 怎么下载？
如果你还不能访问 GitHub，可以：
1. 让能翻墙的同学帮你下载
2. 搜国内镜像站
3. 先用免费试用翻墙，再下载正式版
:::

### 安装

下载后双击安装，一路点下一步就行。

## 第二步：购买机场订阅

"机场"就是提供代理节点的服务商。你需要购买一个机场的订阅。

### 怎么选机场？

- **稳定性** — 最重要的，经常断线的没法用
- **速度** — 看 YouTube、用 AI 都需要速度
- **流量** — 一般每月 100-500GB 足够
- **价格** — 一般 15-50 元/月

::: warning 注意
- 不要买太便宜的（几块钱一个月的大概率不稳定）
- 不要在淘宝买（容易被封）
- 建议找朋友推荐，或者去相关论坛/群组了解
:::

### 购买后你会得到什么？

购买后，机场会给你一个**订阅链接**（一串 URL），类似这样：

```
https://xxx.example.com/api/v1/client/subscribe?token=xxxxxx
```

这个链接就是你的"钥匙"，把它复制下来。

## 第三步：导入订阅

1. 打开你的代理客户端
2. 找到「订阅」或「Profiles」页面
3. 粘贴你的订阅链接
4. 点击导入
5. 等待下载完成，你会看到节点列表

## 第四步：开启代理

1. 找到「代理」页面，选择一个节点
2. 开启「系统代理」
3. 打开浏览器试试能不能访问 Google

::: tip 成功标志
开启后，打开 [Google](https://www.google.com) 试试看！能打开就说明配置成功了。
:::

## 常见问题

### Q：开了代理但上不了外网？
- 检查订阅是否过期
- 换一个节点试试
- 检查系统代理是否开启

### Q：开了代理国内网站变慢？
- 找到「规则模式」（Rule）
- 确保选的是「规则模式」而不是「全局模式」
- 规则模式下，国内网站走直连，外网走代理

### Q：客户端打不开？
- 可能被杀毒软件拦截了，加到白名单
- 右键以管理员身份运行

### Q：怎么开机自动启动？
- 在客户端设置里找到「开机自启」，打开就行
