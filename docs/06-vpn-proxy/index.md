# 翻墙教程

## 什么是"墙"？

中国的网络防火墙（俗称"墙"）屏蔽了一些国外网站，包括：
- **Google** — 全球最好用的搜索引擎
- **YouTube** — 全球最大的视频平台
- **ChatGPT / Claude** — 最强 AI 工具
- **GitHub** — 程序员必备的代码平台
- **Twitter/X** — 学术圈、科技圈都在用

---

## 什么是"翻墙"？

"翻墙"就是通过技术手段绕过这个屏蔽，访问被墙的网站。

### 原理简介

```
你的电脑 → 代理服务器（国外） → 被墙的网站
```

代理服务器帮你"中转"一下，让网站以为你是从国外访问的。

### VPN vs 代理

| | VPN | 代理 |
|--|-----|------|
| 原理 | 所有流量都走加密隧道 | 只有部分流量走代理 |
| 速度 | 一般较慢 | 更快 |
| 用途 | 企业办公、安全连接 | 日常翻墙 |

我们日常说的"翻墙"，用的基本都是**代理**。

---

## 你需要准备什么？

1. **一个代理客户端**（电脑端/手机端各一个）
2. **一个机场订阅**（提供节点的服务）

### 什么是"机场"？

"机场"就是卖代理服务的商家。你付钱，他们给你一个"订阅链接"，导入客户端就能用了。

### 什么是"节点"？

节点就是代理服务器的位置，比如香港、日本、美国等。不同节点速度不同，一般选离你近的（香港、日本）。

### 翻墙合法吗？

- 翻墙本身不违法，但使用翻墙工具访问违法内容是违法的
- 大学生用翻墙来学习、查资料、用 AI 工具，完全没问题

## 电脑端配置

### 整体流程

```
下载客户端 → 购买机场订阅 → 导入订阅 → 开启代理 → 上网！
```

### 第一步：下载代理客户端

你需要一个代理客户端软件。具体用哪个可以问你的机场服务商，他们一般会推荐配套的客户端。

**下载方式**
- **GitHub** — 大部分客户端在 GitHub 上发布
- **机场官网** — 很多机场会提供下载链接
- **朋友分享** — 让已经翻墙的同学发给你

### 第二步：购买机场订阅

"机场"就是提供代理节点的服务商。

**怎么选机场？**
- **稳定性** — 最重要的，经常断线的没法用
- **速度** — 看 YouTube、用 AI 都需要速度
- **流量** — 一般每月 100-500GB 足够
- **价格** — 一般 15-50 元/月

> ⚠️ 不要买太便宜的（几块钱一个月的大概率不稳定）。不要在淘宝买（容易被封）。建议找朋友推荐。

### 第三步：导入订阅

1. 打开你的代理客户端
2. 找到「订阅」或「Profiles」页面
3. 粘贴你的订阅链接
4. 点击导入
5. 等待下载完成，你会看到节点列表

### 第四步：开启代理

1. 找到「代理」页面，选择一个节点
2. 开启「系统代理」
3. 打开浏览器试试能不能访问 Google

> 💡 成功标志：打开 Google 试试看！能打开就说明配置成功了。

## 手机端配置

### iOS（iPhone/iPad）

**获取方式**
1. 注册一个外区 Apple ID（美区、港区等）
   - 搜"注册美区 Apple ID"教程
2. 登录外区 Apple ID
3. 在 App Store 搜索代理客户端
4. 购买并下载

**配置步骤**
1. 打开代理客户端
2. 添加订阅（粘贴你的机场订阅链接）
3. 选择一个节点
4. 打开连接开关
5. 首次使用会提示添加 VPN 配置，点「允许」

### Android

**下载方式**
- **GitHub** — 搜 Android 代理客户端
- **机场官网** — 很多机场提供下载链接

**配置步骤**
1. 打开代理客户端
2. 找到「配置」或「Profiles」
3. 添加订阅（粘贴你的机场订阅链接）
4. 保存后选择一个节点
5. 点击启动

### 节点选择建议

| 地区 | 延迟 | 适合用途 |
|------|------|---------|
| 香港 | 低 | 日常使用首选 |
| 日本 | 低 | 看 YouTube、日语学习 |
| 新加坡 | 低 | 通用 |
| 美国 | 高 | 访问美国专属内容 |

> 💡 日常使用选香港/日本/新加坡（延迟低，速度快）。

## 常见问题

### Q：开了代理但上不了外网？
- 检查订阅是否过期
- 换一个节点试试
- 检查系统代理是否开启

### Q：开了代理国内网站变慢？
- 找到「规则模式」（Rule）
- 确保选的是「规则模式」而不是「全局模式」
- 规则模式下，国内网站走直连，外网走代理

### Q：客户端打不开？
- 可能被杀毒软件拦截了，加到白名单
- 右键以管理员身份运行

### Q：手机和电脑可以用同一个订阅吗？
- 可以！大多数机场允许 3-5 个设备同时在线

### Q：连接后有些 App 还是打不开？
- 尝试切换节点
- 在客户端里开启「全局模式」试试
