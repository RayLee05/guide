## 电脑端配置

### 流程

```
下载客户端 → 注册机场 → 导入订阅 → 开启代理 → 上网
```

### 步骤

1. **下载客户端** — 问机场商推荐，或去 [GitHub](https://github.com) 搜
2. **注册机场** — 选上面的机场注册，购买订阅
3. **导入订阅** — 客户端 → 订阅 → 粘贴链接 → 导入
4. **开启代理** — 代理页面 → 选节点 → 开启系统代理

> 💡 成功标志：能打开 [Google](https://google.com)

## 手机端配置

### iOS

1. 注册外区 Apple ID（搜"注册美区 Apple ID"）
2. App Store 搜代理客户端，购买下载
3. 添加订阅 → 选节点 → 开启连接

### Android

1. 去 GitHub 或机场官网下载 APK
2. 添加订阅 → 选节点 → 启动

## 常见问题

| 问题 | 解决方法 |
|------|---------|
| 开代理上不了外网 | 检查订阅是否过期，换节点 |
| 国内网站变慢 | 切换到「规则模式」 |
| 手机电脑能用同一订阅 | 可以，一般支持 3-5 台设备 |
