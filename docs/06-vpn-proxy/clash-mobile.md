# 手机端翻墙配置

## iOS（iPhone/iPad）

### 下载客户端

iOS 上的代理客户端一般需要付费（几美元），而且在国区 App Store 下载不到。

#### 获取方式

1. **注册一个外区 Apple ID**（美区、港区等）
   - 搜"注册美区 Apple ID"教程
   - 关键：地址填国外，支付方式选"无"
2. 登录外区 Apple ID
3. 在 App Store 搜索代理客户端
4. 购买并下载

::: tip 省钱技巧
注册外区 Apple ID 后，可以买 App Store 礼品卡来充值，不需要国外信用卡。
:::

#### 配置步骤

1. 打开代理客户端
2. 添加订阅（粘贴你的机场订阅链接）
3. 选择一个节点
4. 打开连接开关
5. 首次使用会提示添加 VPN 配置，点「允许」

### Android

#### 下载客户端

Android 用户可以直接下载 APK 安装。

##### 下载方式

- **GitHub** — 搜 Android 代理客户端
- **机场官网** — 很多机场提供下载链接
- **Google Play** — 如果能访问的话

##### 配置步骤

1. 打开代理客户端
2. 找到「配置」或「Profiles」
3. 添加订阅（粘贴你的机场订阅链接）
4. 保存后选择一个节点
5. 点击启动

## 节点选择建议

| 地区 | 延迟 | 适合用途 |
|------|------|---------|
| 香港 | 低 | 日常使用首选 |
| 日本 | 低 | 看 YouTube、日语学习 |
| 新加坡 | 低 | 通用 |
| 台湾 | 中 | 中文内容 |
| 美国 | 高 | 访问美国专属内容 |

::: tip 选节点技巧
- 日常使用选**香港/日本/新加坡**（延迟低，速度快）
- 看 Netflix 美区内容选**美国**
- 用 AI 工具选**不限制的地区**（有些节点会被 AI 服务封禁）
:::

## 常见问题

### Q：iOS 提示「无法验证 App」？
- 设置 → 通用 → VPN 与设备管理 → 找到对应的开发者 → 信任

### Q：Android 安装提示「未知来源」？
- 设置 → 安全 → 允许安装未知来源应用

### Q：手机和电脑可以用同一个订阅吗？
- 可以！大多数机场允许 3-5 个设备同时在线

### Q：连接后有些 App 还是打不开？
- 尝试切换节点
- 在客户端里开启「全局模式」试试
- 有些 App 有地区限制，需要特定地区的节点
