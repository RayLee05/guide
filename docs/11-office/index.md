## Word 常用功能

| 功能 | 快捷键 |
|------|--------|
| 加粗/斜体/下划线 | Ctrl+B / Ctrl+I / Ctrl+U |
| 查找替换 | Ctrl+H |
| 保存 | Ctrl+S |

**写论文**
- 用「标题 1」「标题 2」样式 → 可自动生成目录
- 引用 → 目录 → 自动目录

## Excel 常用函数

| 函数 | 用途 | 示例 |
|------|------|------|
| SUM | 求和 | =SUM(A1:A10) |
| AVERAGE | 平均值 | =AVERAGE(A1:A10) |
| IF | 条件 | =IF(A1>60,"及格","不及格") |
| VLOOKUP | 查找 | =VLOOKUP(A1,Sheet2!A:B,2,0) |

## PPT 原则

1. **简洁** — 字号 ≥ 24pt，不要放太多字
2. **统一** — 同一主题/配色/字体
3. **视觉化** — 能用图就不用文字

**模板**：新建时搜索、[canva.cn](https://www.canva.cn)

## 常见问题

| 问题 | 解决方法 |
|------|---------|
| Word 怎么保存 | Ctrl+S，建议存 D 盘 |
| Excel 数字变日期 | 先输入单引号 ' 再输入数字 |
| PPT 好丑 | 用模板！新建时搜"模板" |
| 文件打不开 | 重新下载，或用 WPS 打开 |
